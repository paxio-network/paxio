# M-L10.2 — CSS tokens + theme system + fonts

**Тип**: Frontend implementation (frontend-dev).
**Зависимости**: M-L10 P1 (foundation merged), `docs/design/paxio-b5/EXTRACTED.md` доступен.
**Worktree**: `/tmp/paxio-ml10.2-css` (frontend-dev).

**Estimated**: 2 hours.

## Цель

Port CSS tokens из `docs/design/paxio-b5/EXTRACTED.md` в `apps/frontend/landing/app/globals.css`.
Установить theme system (light/dark) + Google Fonts через `next/font/google`. Без Hero,
без Scrolls, без Header — только token foundation.

## Готово когда

1. `apps/frontend/landing/app/globals.css` содержит **обе** темы (light defaults + dark
   overrides via `html[data-theme="dark"]`):
   - 5 paper tokens (`--paper-0..3`)
   - 5 ink tokens (`--ink-0..4`)
   - 3 gold tokens (`--gold`, `--gold-bright`, `--gold-ink`)
   - 3 semantic (`--up`, `--down`, `--blue`)
   - 3 line tokens (`--line`, `--line-soft`, `--line-xsoft`)
   - 3 font-family vars (`--f-display`, `--f-sans`, `--f-mono`)
   - 2 layout vars (`--page-pad`, `--ease-out`)
2. Body BG = `var(--paper-0)`, color = `var(--ink-1)`, font = `var(--f-sans)`,
   font-size 15px, line-height 1.55, paper grain via inline SVG noise data-URI
   (см. `EXTRACTED.md` "Misc effects" section).
3. `apps/frontend/landing/app/layout.tsx` импортирует через `next/font/google`:
   - Fraunces (опции: 400/500/600/700, opsz 9..144, italic+roman) → CSS var `--f-display`
   - Inter Tight (400/500/600/700) → `--f-sans`
   - JetBrains Mono (400/500/700) → `--f-mono`
4. `<html data-theme={theme}>` устанавливается клиентом — создай
   `app/_components/ThemeProvider.tsx` ('use client') который:
   - Читает `localStorage.getItem('paxio-theme')` на mount
   - Default: `'light'` если ничего нет
   - Устанавливает `<html>` атрибут
   - Экспортирует hook `useTheme()` → `[theme, setTheme]`
5. `globals.css` имеет primitives из EXTRACTED.md "Component contracts":
   `.btn`, `.btn.solid`, `.btn.gold`, `.btn.ghost`, `.btn.sm`,
   `.chip`, `.chip.gold`, `.chip.ink`, `.chip-dot`,
   `.panel`, `.panel.raised`, `.panel.paper-2`, `.panel-head`,
   `.kicker`, `.mono`, `.serif`, `.italic`, `.scribble`,
   `.kv`, `.kv .k`, `.kv .v`,
   `.pulse-dot` + `@keyframes pulse`,
   `.blink` + `@keyframes blink`,
   `.marquee` + `.marquee-track` + `@keyframes marquee`,
   `.h-xl`, `.h-lg`, `.h-md`, `.h-sm`,
   `.grid-12`, `.col-span-{4,5,6,7,8,12}`,
   `.page`, `section { padding-block: 88px }`, `section.tight`,
   `.hand`, `.hand.soft`, `.reveal`, `.reveal.in`,
   `[data-motion="off"]` overrides
6. `pnpm --filter @paxio/landing-app build` проходит clean
7. `pnpm typecheck` clean
