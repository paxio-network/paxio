**Fix:** `apps/frontend/landing/app/data/preview.ts`:
- line 34: `name:'escrow.paxio'` → `name:'btc-escrow.paxio'`
- line 35: `name:'dca.paxio'` → `name:'btc-dca.paxio'`
- line 111 + 309 (где встречаются ещё ссылки) — same revert
- M-L10.4 hero-b5.test.tsx assertion на ll 149 уже expects `'escrow.paxio'`/`'dca.paxio'`
  — architect обновит test assertions to match.

### D-2. Disable dark theme (per user request)

User: «дарк просто отключи». Удалить `html[data-theme="dark"]` block из globals.css.
ThemeProvider оставить (не удалять прокачку через React) — но CSS overrides убрать,
так что переключение дарк больше не имеет визуального эффекта.

**Fix:** `apps/frontend/landing/app/globals.css`:
- Удалить весь блок `html[data-theme="dark"] { ... }` (~30-50 строк).
- Пустую строку или комментарий `/* dark theme disabled per user — landing
  light-only until further notice */` оставить как marker.

### D-3. State strip typography — bold tabular numbers vs italics

Target (`shoud-be-01.png`): big numbers в state strip (`2,483,983`, `42.3%`, `$10.2M`,
`682 +0% / -0% / etc.`) — **bold**, **tabular-nums**, **mono font**, no italics.
Italics только на small descriptive text («agents indexed across 6 registries»).

Current (`vercel_light_01.png`): everything is **italics**, weights uniform, numbers
not displayed как монослойный bold. Слабо читается.

**Fix:** `apps/frontend/landing/app/sections/01-hero-b5.tsx::HeroB5::State of Economy
strip block`:
- Big numbers — wrap в `<b style={{ fontFamily: 'var(--f-mono)', fontVariantNumeric:
  'tabular-nums', fontWeight: 700 }}>` (or use existing `.mono` class with bold variant).
- Verify state-strip CSS class в hero-variants.css applies `.state-text b { font-variant-numeric: tabular-nums; font-weight: 700; }` — possibly already there, just ensure component applies `<b>` to numbers consistently.

### D-4. PaeiTicker — stronger dotted dividers + prominent numbers

Target (`shoud-be-01.png`): PaeiTicker row после state strip — 6 cells separated by
**stronger dotted dividers**, numbers prominent (font-size ~14-16px, weight 600-700,
mono).

Current: dividers слабые, numbers smaller, weight light.

**Fix:** `01-hero-b5.tsx::TickerCell` component:
- Adjust inline style `borderRight: '1px dashed rgba(26,22,18,0.18)'` →
  `'1px dashed rgba(26,22,18,0.42)'` (stronger).
- Number rendering: bump fontSize 10 → 13, fontWeight 400 → 700, fontFamily mono.

### D-5. Doors section — 4 columns, NOT 2x2 grid + add stats banner

Target (`shoud-be-13.png`): «Pick your door» section — **4 cards в одной горизонтальной
строке** (Builder / Buyer / Analyst / Enterprise). НИЖЕ — dark stats banner with «Last
24h on Paxio: 1.22M / 48,291 / $20.7M / 100%».

Current (`vercel_light_13.png`): 2x2 grid + headline «Four ways to enter the agentic
economy.» (target headline отсутствует — это inline в block label) + missing stats
banner.

**Fix:** `01-hero-b5.tsx::Doors` (или footer block в page.tsx):
- Grid template: `grid-template-columns: repeat(4, 1fr)` (currently `repeat(2, 1fr)`).
- Удалить/уменьшить «Four ways to enter» большой headline — target имеет только small
  kicker `§ 06 · PICK YOUR DOOR`.
- Добавить dark stats banner ниже doors с numbers from `PREVIEW_NETWORK_SNAPSHOT`
  (txns, agents, valueMoved). Уже есть PREVIEW_NETWORK_SNAPSHOT export.
