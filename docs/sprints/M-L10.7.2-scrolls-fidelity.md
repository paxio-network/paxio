## Конкретные расхождения

### D-1. ScrollSDK kicker text

Source `docs/design/paxio-b5/components/landing_scrolls_b5.jsx::ScrollSDK` использует
kicker "**SDK · BUILDERS**" (или близкий текст с upper-case BUILDERS).
Current impl `02-scrolls-b5.tsx:177` — `kicker="NPM package · developers"`.

**Fix:** изменить kicker на `"SDK · builders"` (или ровно как в design source — frontend-dev сверится с jsdoc/JSX in `docs/design/paxio-b5/components/landing_scrolls_b5.jsx`).

### D-2. ScrollBitcoin headline — Bitcoin not on-chain

Target screenshots (`shoud-be-05.png`) shows headline "**Your agent. _A real Bitcoin
address._**" — слово «Bitcoin» вместо «on-chain».

Current `02-scrolls-b5.tsx:358`: `<span className="serif display-italic">A real on-chain
address.</span>`.

**Fix:** заменить «on-chain address» на «Bitcoin address». Line 465 («✓ on-chain address»
в bullet list внутри code card) — also update to «✓ Bitcoin address» если соответствует
design source code-card content.

### D-3. ScrollRadar headline — "We don't host agents."

Target (`shoud-be-08.png`): headline ровно «**We don't host agents.**» — 3 слова.

Current `02-scrolls-b5.tsx:568`: «We do not host agents.» + дополнительно «We measure
them.» во второй строке (out of design source).

**Fix:** заменить headline на ровно «We don't host agents.» (с апострофом). Если есть
секондарная строка «We measure them.» в impl — удалить.

### D-4. ScrollNetwork — CTA banner стилистика

Target (`shoud-be-11.png`): дark banner-like CTA «**REGISTER YOUR AGENT — JOIN THE
NETWORK →**» с tracking + ALL CAPS + dark background panel.

Current `02-scrolls-b5.tsx:1208-1218`: CTA exists but текст mixed-case «Register your
agent — join the network →» и не имеет dark banner styling — просто inline link.

**Fix:** обновить копи в ALL CAPS («REGISTER YOUR AGENT — JOIN THE NETWORK →»),
добавить class или inline style для dark banner panel (`background: var(--ink-0)`,
`color: var(--paper-0)`, padding generous). Визуально — full-width dark band с
центрированным текстом per target.
