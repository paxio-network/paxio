## Конкретные расхождения

### D-1. Ticker labels — restore PAEI·* prefix

| Cell | Landing (drifted) | Design source |
|---|---|---|
| a | `BTC` | `PAEI·BTC` |
| b | `LEGAL` | `PAEI·LEGAL` |
| c | `FINANCE` | `PAEI·FINANCE` |
| d | `RESEARCH` | `PAEI·RESEARCH` |
| e | `CX` | `PAEI·CX` |
| f-j (extras) | `SECURITY/INFRA/DEFI/LANG/DEV` | `PAEI·SECURITY/PAEI·INFRA/PAEI·DEFI/PAEI·LANG/PAEI·DEV` |
| k | `AGENTS` | `PAEI·AGENTS` |
| l | `PXI COMPOSITE` | (design has 12 PAEI cells; "PXI" branding не существует в design) |

Landing impl — also reorder to match design:
- a = `PAEI` (the composite — was at "l", was labeled `PXI COMPOSITE`)
- b = `PAEI·BTC`
- c..k = sub-indices in design order
- l = `PAEI·AGENTS`

Plus `PREVIEW_TICKER_EXTRA_LANES` in `preview.ts` — добавить `PAEI·` prefix к labels SECURITY/INFRA/DEFI/LANG/DEV.

### D-2. State strip ending — "PAEI" not "PXI Composite"

Landing line 603 (current):
```tsx
PXI Composite <b>{t.paei.toFixed(2)}</b> <span>▲{t.paeiD}%</span>
```

Design source line 178:
```tsx
PAEI <b>{t.paei.toFixed(2)}</b> <span>▲{t.paeiD}%</span>
```

**Fix:** `PXI Composite ` → `PAEI ` in state strip closing token.

### D-3. Remove drift-justification comment

Landing impl carries comment (lines 162-167):
> «ticker labels use plain category names — NOT 'PAEI·LEGAL' — to keep PAEI text
> unique in the DOM for the hero-b5.test.tsx ticker section assertion»

Cited assertion (`hero-b5.test.tsx:172`):
```tsx
const paeiNode = screen.queryByText(/PAEI/i) ?? document.querySelector(...);
expect(paeiNode, '...').toBeTruthy();
```

`queryByText` returns ANY match (single or multiple) — uniqueness NOT required.
The justification was wrong. Remove the comment so future developers don't
re-introduce drift.
